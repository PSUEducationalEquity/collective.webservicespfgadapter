Change History
==============

1.3.0
----------------

- Added an option to use the field IDs as the submission keys
  instead of the field names


1.2.0
----------------

- Implemented all of the extra data fields so they provide values
- Added support for the PFGDataGridField
- Improved support for Likert and multi-select fields
- Increased the timeout value to 60 seconds


1.1.0
----------------

- Submitted data uses OrderedDict to maintain field order in submission
- Add an incrementing number to duplicate field names
- Fieldset titles can be prepended to field names
- Removed some of the extra data that was meaningless
- Improved the extra data descriptions


1.0.0
----------------

- Initial release
